▪ Wireframe Notes — What this design does
HEADERSame masthead as libraryofwar.com. Era pills replace global nav breadcrumb. Record count + mode toggle live in the section bar.
LEFT SIDEBARPersistent filter panel + scrollable soldier list. Name, rank, branch, date of loss. Alphabetical or date sort. Color-coded dot = status. Selected row = accent border.
MAP CENTERFull-height MapLibre panel. Light tiles by default (matches editorial). Cluster bubbles use ink/rule palette. Selected marker uses accent red. On This Day: bottom-left chip.
STATS BARBelow map. Shows aggregate counts for current filter: Total, KIA, MIA, WIA, POW, date range. Updates on filter change. Numbers in Playfair/Georgia large format.
RIGHT PANELService record. Prev/next navigation through filtered list. No photo by default — placeholder shown. Fields: name, rank, branch, unit, date of loss, DOB, location, service number, source link.
FREE DATAAll fields shown are public domain: NARA records, VVMF Vietnam Memorial Wall database, DoD casualty records. No hometown addresses, no next-of-kin, no photos.
STATUS COLORSKIA = red · MIA = amber · WIA = blue · POW = orange. Consistent across list dot, map marker, pill badge, and stat number.
