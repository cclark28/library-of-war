04 — Homepage Logic

Articles without images excluded from featured sections

Hero grid, Latest Dispatches, and From the Archive now only pull from articles with a mainImage. Articles without images still appear in era archive sections, browse pages, category pages, and search — fully linked, just not featured. Applied automatically going forward via src/app/page.tsx.

05 — Build Fix

Sanity schema TypeScript error resolved

sanity/schemas/article.ts had a prepare() function with a typed destructure incompatible with Sanity's Record<string, any> input type. Fixed by accepting the raw value object and destructuring internally with a null-safe label lookup.
